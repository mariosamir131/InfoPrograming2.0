Java



¿Qué es Java?

Java es una plataforma informática y a su vez un lenguaje de programación creado en 1995 por la empresa Sun Microsystem. El objetivo de este lenguaje es que los programadores sólo tuvieran que escribir el código de un programa una vez, y que éste, pudiese ejecutarse en cualquier dispositivo. Esto es posible gracias a la Máquina Virtual de Java (JVM), que brinda esa portabilidad necesaria. Con Java se pueden crear programas en una gran variedad de dispositivos, permitiendo ejecutar la misma aplicación en diversos sistemas operativos. El nombre inicial con el que se iba a denominar era Oak, pero al estar la marca registrada se optó por Java.


Programación en Java

Java es un lenguaje orientado a objetos, independiente de la plataforma hardware donde se desarrolla, y que utiliza una sintaxis similar a la de C++ pero reducida. Es un lenguaje con una curva de aprendizaje baja (se puede decir que es fácil de aprender) y que dispone de una gran funcionalidad de base (incrementada por la gran cantidad de código de terceros existente). Java, como lenguaje de programación, ofrece un código robusto, que ofrece un manejo automático de la memoria, lo que reduce el número de errores. La comunidad de programadores Java existente es muy extensa, en torno a los 9 millones en todo el mundo, y muy activa, lo que genera una gran cantidad de recursos actualizados. Existe un lenguaje de programación llamado Javascript que se presta a confusión por su similitud en el nombre con el de Java. A pesar del parecido en el nombre, estamos ante dos lenguajes de programación totalmente diferente: mientras que Java es un lenguaje orientado a objetos, Javascript es un lenguaje de programación que se basa en objetos.


¿Para qué sirve Java?

A pesar de que se suele hablar de JavaScript como un lenguaje que está diseñado principalmente para el desarrollo web front-end y en efecto, es en lo que me he enfocado hasta el momento - la respuesta a la pregunta "¿Para qué sirve Java?" no es tan simple. Junto a elementos interactivos web, un conocimiento amplio de JavaScript te permitirá crear las siguientes cosas:

Videojuegos
Si estás interesado en videojuegos de internet y desarrollo de videojuegos, deberías considerar usar tus conocimientos en JavaScript para crear tus propios videojuegos basados en el navegador. Al mismo tiempo existen limitaciones que rodean la complejidad de los videojuegos de navegador, JavaScript es tan bueno como cualquier otro lenguaje de programación cuando se trata de crearlos. Crear tus propios videojuegos es también una forma de practicar tus conocimientos para programar en Java. Averiguar las respuestas a preguntas como "¿Para qué sirve Java?" no siempre es fácil, pero pienso que es la mejor manera para averiguar exactamente qué tan versátil puede ser programar en Java tratando de realizar cosas por ti mismo. Descarga un editor de programación (mencionare este tema más tarde) e inventa una idea para un juego simple y utiliza un poco de tu tiempo para crearlo.

Aplicaciones móviles

La mayoría del desarrollo de las aplicaciones móviles están realizados con un sistema operativo específico de lenguaje como Swift (iOS) o Java (Android), por lo que no hay razón para no crear tus propias aplicaciones con JavaScript. La implementación de frameworks como Phonegap y React Native hace posible crear aplicaciones móviles para múltiples sistemas operativos utilizando el mismo código. En efecto, JavaScript parece una gran opción para el desarrollo de aplicaciones móviles en el futuro.

Web y Aplicaciones de servidor

Nuevamente, la respuesta más común a la pregunta "¿Para qué sirve Java?" es algo como "crear elementos interactivos web". Pero eso no es todo. La creación de nuevas librerías y frameworks permite a los desarrolladores web crear gran cantidad de programas Back-end con JavaScript. Esto incluye cosas como aplicaciones web y aplicaciones de servidor. En efecto, JavaScript se convirtió en una herramienta esencial para desarrolladores web back-end como para desarrolladores front-end. Tienes que tener en cuenta que esto no es de ninguna manera una lista exclusiva. JavaScript es un lenguaje de programación muy poderoso y la mayoría de las cosas solo estarán limitadas por tu imaginación.
